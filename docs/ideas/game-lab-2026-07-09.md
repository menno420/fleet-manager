# game-lab — two-track GBA game venture (pokeemerald mod + Butano homebrew)

> **Status:** `ideas`

**Idea:** a gen-2 lane shipping playable GBA games on two tracks sharing one
toolchain and one headless test loop — **Track A:** a private Pokémon Emerald
mod on the pret/pokeemerald decomp (PRIVATE repo only — Nintendo-copyrighted
material, never published); **Track B:** original public GBA homebrew on
Butano (publish-safe by construction, external publishing owner-gated).
**Why worth having:** the usual killer for a games lane — toolchain risk —
was retired the same night by the toolchain scout session
([`../findings/gba-toolchain-proof-2026-07-09.md`](../findings/gba-toolchain-proof-2026-07-09.md)):
byte-identical pokeemerald builds (1m20s full / 2.0s incremental), Butano
via the leseratte10 devkitARM r68 mirror (17.5s), and a scripted mGBA
headless verify loop (~290 fps, in-game mod verification with zero human
input) — so the lane starts at product work, not infrastructure archaeology,
and gives the owner the most directly *playable* artifact the fleet produces.
**Unblocks:** the second gen-2 born-right launch after venture-lab; a
creative/entertainment surface for the fleet; reusable emulator-driven
verification patterns for any future game lane.
