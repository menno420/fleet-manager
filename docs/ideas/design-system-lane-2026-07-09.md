# Design-system lane — shared visual kit across the fleet

> **Status:** `ideas`

**Idea:** one shared visual kit (SVG assets, HTML/CSS components, palette,
type) used across websites, bot embeds/images, and future apps — a lane
that owns it and PRs it into consumers.
**Why worth having:** every surface currently invents its own look; a
shared kit makes the fleet's output read as one product family and stops
each lane re-solving design from zero.
**Unblocks:** faster, better-looking frontends for venture-lab and the
mobile-lab idea; a consistent brand the devlog lane can point at.
