# Mobile-lab lane — working mobile apps via React Native + Expo

> **Status:** `ideas`

**Idea:** a gen-2 lane that builds working mobile apps with React Native +
Expo — QR-code testing on the owner's phone, EAS cloud builds (PWA as the
fastest alternative when native isn't needed); the owner is the device-tester,
and store publishing is the owner-gated last mile.
**Why worth having:** it extends the fleet from web/bot surfaces to the one
platform the owner uses most, with a test loop (scan a QR, hold the app) a
non-coder can drive personally.
**Unblocks:** a candidate second gen-2 launch after venture-lab; mobile
frontends for anything venture-lab validates.
