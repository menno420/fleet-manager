### Conclusions

1. **Ack rate 2/9.** Both acks came from sessions that happened to be live
   mid-mission and re-read the inbox; **zero** of the 7 lanes without a live
   session acked, hours later. NO ACK here is a *liveness* result, not a
   protocol failure ‚ÄĒ the loop's read half runs only when something wakes it.
2. **Read latency is bimodal.** With a live session: 9m47s / 14m43s
   dispatch‚Üíack-on-main (and superbot-next *discovered* the order in 7m32s by
   reading the manager's open PR branch ‚ÄĒ ack landed on main 4m10s before the
   order did, since the ack rode a check-less path while the order waited out
   6 required checks). Without a live session: **unbounded**.
3. **Wake routines are the missing half of the coordination loop** ‚ÄĒ this
   measures what the blueprint asserted (¬ß3.6 "routines are the highest-value
   click"). Order-pickup SLA ‚Čą routine cadence + ~15 min (in-session read‚Üíack
   cost measured 2‚Äď11 min, plus boot). Concrete per-class cadences now in
   `../gen2-blueprint.md` ¬ß2a.
4. **Awake-but-missed is a real class (1/9):** codetool-lab-opus4.8 committed
   a status heartbeat 15m31s after the ping landed without acking it ‚ÄĒ the
   session never re-read the inbox before its final status write. Gen-2
   template delta #5/#10 ("re-read the inbox at HEAD immediately before the
   final status write") is now measured, not theoretical.
5. **Self-reported `updated:` stamps are unreliable ‚ÄĒ derive timing from the
   commits API (R2 in miniature).** mining's status says `18:00Z` but its
   content was authored pre-17:55 and committed 19:02:46Z; exploration's says
   `18:05Z` but the file's last commit is 17:06:06Z. Pattern consistent with
   local-time-written-as-Z (+1h) drift. Gen-2 template: timestamps come from
   `date -u`, never from the model's sense of time.
6. **Owner-gated lanes cannot ack even when they work** (mining's only status
   channel was a PR the owner merged 3h later). A lane's status write must
   ride a path the lane itself can merge ‚ÄĒ the control fast lane +
   self-merge-on-green grant in the gen-2 seed exist for exactly this.
